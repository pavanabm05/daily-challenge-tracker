Daily Challenge & Streak Tracker

Overview

Daily Challenge & Streak Tracker is a simple website that gives users a random daily challenge and tracks their completion streak. It helps users stay productive and build good habits.

Features

- Random daily challenge generator
- Completed button to mark task done
- Streak counter to track consistency
- Progress stored using Local Storage

Technologies Used

HTML, CSS, JavaScript

How It Works

When the website loads, it shows a random challenge. The user completes the task and clicks the Completed button. The website then updates the user's streak and saves the progress in the browser.

Future Improvements

- Challenge categories
- Skip or refresh challenge option
- Progress dashboard
